Kubernetes Commands

Command Description Example
kubectl get pods List all pods in the current namespace kubectl get pods
kubectl get deployments List all deployments in the current namespace kubectl get deployments
kubectl describe pod [pod-name] Show detailed information about a specific pod kubectl describe pod nginx-pod
kubectl logs [pod-name] Print the logs for a container in a pod kubectl logs nginx-pod
kubectl apply -f [filename] Create/update resources from a YAML file kubectl apply -f deployment.yaml
kubectl delete pod [pod-name] Delete a specific pod kubectl delete pod nginx-pod
kubectl exec -it [pod-name] -- [command] Execute command in a container kubectl exec -it nginx-pod -- /bin/bash
kubectl port-forward [pod-name] [local-port]:[pod-port] Forward a local port to a port on the pod kubectl port-forward nginx-pod 8080:80

RHEL Commands

Command Description Example
rpm -qa List all installed RPM packages rpm -qa | grep httpd
dnf install [package] Install a package using DNF dnf install httpd
systemctl start [service] Start a service systemctl start httpd
systemctl enable [service] Enable a service to start at boot systemctl enable httpd
firewall-cmd Configure the firewall firewall-cmd --add-service=http --permanent
journalctl -u [service] View logs for a service journalctl -u httpd
subscription-manager list List subscription status subscription-manager list --installed

PowerShell Commands

Command Description Example
Get-Command Get all commands available in PowerShell Get-Command -Module Microsoft.PowerShell.Management
Get-Help Get help on a command Get-Help Get-Process -Detailed
Get-Process Get all running processes Get-Process | Sort-Object CPU -Descending
Get-Service Get all services Get-Service | Where-Object {$_.Status -eq "Running"}
Invoke-Command Run a command on a remote computer Invoke-Command -ComputerName Server01 -ScriptBlock {Get-Process}
New-Item Create a new item (file, directory, etc.) New-Item -Path C:\temp -Name test.txt -ItemType File
Start-Service Start a Windows service Start-Service -Name BITS

Command Description Example

terraform validate Validate the configuration files terraform validate

Docker Commands

Command Description Example
docker run Run a container from an image docker run -d -p 80:80 nginx
docker ps List running containers docker ps -a
docker build Build an image from a Dockerfile docker build -t myapp:1.0 .
docker images List available images docker images
docker exec Execute a command in a running container docker exec -it container_id bash
docker logs Fetch the logs of a container docker logs container_id
docker-compose up Create and start containers defined in docker-compose.yml docker-compose up -d
